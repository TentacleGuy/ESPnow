- Links auf das PlatformIO Symbol klicken
- Create New Project
![[Pasted image 20260625221826.png]]
 - Passendes Board auswählen(in meinem Fall der ESP32S3-R8N16)
![[Pasted image 20260625222143.png]]
- Framwork: Arduino
- (optional) Speicherort auwählen
- jetzt sollte euer Ordner so aussehen:
![[Pasted image 20260625222342.png]]
-  Ordner  in src für receiver und transmitter anlegen um beide Geräte in einem Projekt zu verwalten(kann aber auch ein Projekt pro Gerät erstellt werden)
	![[Pasted image 20260625223613.png]]
	platformio.ini anpassen:
	```
	[env:receiver]
	platform = espressif32
	board = 4d_systems_esp32s3_gen4_r8n16
	framework = arduino
	build_src_filter = +<receiver/> -<transmitter/>
	upload_port = COM5
	upload_speed = 115200
	monitor_port = COM5
	monitor_speed = 115200
	lib_deps =
	
	[env:transmitter]
	platform = espressif8266
	board = d1_mini
	framework = arduino
	build_src_filter = +<transmitter/> -<receiver/>
	upload_port = COM6
	upload_speed = 115200
	monitor_port = COM6
	monitor_speed = 115200
	lib_deps =
	```
- wenn du nur ein Board pro Projekt machst, musst du nur einen Block angeben und keine unterordner erstellen, dann kommt alles direkt in den src Ordner
- Ports und Boards ggf. anpassen
